# AI Cursor Setup

## Project Overview

This project demonstrates how to set up an AI-assisted development environment using Cursor IDE and AI tools.
A simple landing page was created using HTML and CSS with the help of AI-generated suggestions.

The goal of this project was to explore how AI tools can assist developers in writing and improving code efficiently.

---

## Tools Used

* Cursor IDE
* GitHub Copilot
* ChatGPT
* Claude AI
* GitHub

---

## Development Environment Setup

### Cursor IDE

Cursor IDE was installed and configured as the primary development environment.
It was used to create and edit the project files with AI-assisted coding.

### GitHub Copilot

GitHub Copilot was connected to the IDE and tested inside the project.
It provided automatic code suggestions while writing HTML and CSS.

### ChatGPT

ChatGPT was used to generate example functions and explanations for comparison with Claude AI.

### Claude AI

Claude AI was used to test prompt engineering and generate code responses for comparison.

---

## Prompts Used

The following prompts were used with AI tools to generate and improve the project:

1. Create a simple landing page using HTML and CSS with a navigation bar, title, paragraph, and button.
2. Create a navigation bar with Home, About, and Contact links using HTML and CSS.
3. Update the "Get Started" button so that it scrolls smoothly to the Features section.
4. Create a footer with copyright text and a privacy policy link.

---

## Project Features

The landing page includes the following components:

* Navigation bar with Home, About, and Contact links
* Hero section with title and description
* "Get Started" button with smooth scrolling
* Features section explaining AI tools
* Footer with copyright and privacy link

---

## ChatGPT vs Claude Comparison

Prompt used:

Write a JavaScript function to check if a number is prime.

### ChatGPT Response

* Simple and beginner-friendly code
* Clear explanation of logic
* Easy to understand for new developers

### Claude Response

* More optimized code
* Uses efficient logic (square root method)
* Provides detailed explanation

### Conclusion

Both tools generate correct code, but their approaches differ slightly.
ChatGPT focuses more on simplicity, while Claude often provides optimized logic and more detailed explanations.
